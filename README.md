# 面包店菜单展示应用

一个使用React构建的面包店每日菜单展示网页应用，具有响应式设计和图片展示功能。

## 功能特性

- 🍞 展示面包店每日菜单及价格
- 📱 响应式设计，适配移动端和桌面端
- 🖼️ 图片展示功能（点击查看面包实物图）
- 🎨 美观的UI设计，使用Amber色系主题
- 🔔 通知系统
- 📋 复制链接功能

## 技术栈

- **React 18** - 前端框架
- **React DOM** - React渲染
- **Tailwind CSS** - 实用类CSS框架
- **Lucide React** - 图标库
- **rc-notification** - 通知组件
- **rc-dialog** - 对话框组件

## 使用方法

1. 直接打开 `index.html` 文件即可运行
2. 或部署到任何静态网页托管服务

## 页面截图

- **主菜单页面**：展示所有面包产品及价格
- **图片展示模态框**：点击"实物图展示"按钮查看面包图片
